# DSL4Tree
## Installation
### Requirements:
* Python3
* [textX](http://textx.github.io/textX/stable/) v2.1
```
$ pip install textX
```
* [matplotlib](https://matplotlib.org/) v3.2.1
```
$ pip install matplotlib
```
* [Pillow](https://pillow.readthedocs.io/en/stable/) v7.1.1
```
$ pip install pillow
```
## Usage
```
python3 main.py
```
